Questions fréquentes

Quel est le délai de production ?

Les délais varient selon le type de carte : 5 jours pour les cartes standard, 7 jours pour les premium et 10 jours pour les cartes NFC.

Puis-je modifier mes cartes après commande ?

Oui, pour les cartes NFC, vous pouvez modifier les informations à tout moment via votre espace client. Pour les autres cartes, les modifications sont possibles avant la production.

Comment fonctionnent les cartes NFC ?

Les cartes NFC contiennent une puce qui peut être lue par les smartphones compatibles. En approchant la carte du téléphone, vos informations de contact sont automatiquement partagées.

Proposez-vous des échantillons ?

Oui, nous pouvons vous envoyer un kit d'échantillons contenant nos différents types de cartes. Contactez-nous pour en faire la demande.
